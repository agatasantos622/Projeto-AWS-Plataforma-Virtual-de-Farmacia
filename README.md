# 💊 Projeto AWS - Plataforma Virtual de Farmácia

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de criar uma proposta de infraestrutura em nuvem para uma farmácia virtual fictícia chamada **Abstergo Industries**, utilizando serviços da AWS (Amazon Web Services).

A proposta busca demonstrar como a computação em nuvem pode ajudar empresas a reduzir custos operacionais, aumentar a escalabilidade e melhorar a segurança dos dados.

---

# ☁️ Serviços AWS Utilizados

## 🔹 Amazon EC2

Responsável pela hospedagem da aplicação web da farmácia virtual.

### Benefícios:

* Escalabilidade sob demanda
* Redução de custos com servidores físicos
* Alta disponibilidade

---

## 🔹 Amazon S3

Utilizado para armazenamento de arquivos, imagens de produtos, backups e documentos.

### Benefícios:

* Armazenamento seguro
* Alta durabilidade
* Redução de custos com armazenamento local

---

## 🔹 Amazon RDS

Responsável pelo gerenciamento do banco de dados da aplicação.

### Benefícios:

* Backups automáticos
* Monitoramento simplificado
* Maior segurança e desempenho

---

# 🎯 Objetivos do Projeto

* Reduzir custos operacionais
* Modernizar a infraestrutura da empresa
* Melhorar a disponibilidade da plataforma
* Garantir maior segurança dos dados
* Aplicar conceitos de computação em nuvem na prática

---

# 🏗️ Arquitetura da Solução

A infraestrutura da plataforma foi organizada da seguinte forma:

```text
Cliente → Amazon EC2 → Amazon RDS
                 ↓
            Amazon S3
```

---

# 📊 Comparativo de Custos

| Infraestrutura Tradicional | Solução AWS          |
| -------------------------- | -------------------- |
| Servidores físicos         | Amazon EC2           |
| Armazenamento local        | Amazon S3            |
| Banco de dados manual      | Amazon RDS           |
| Alto custo de manutenção   | Serviços gerenciados |
